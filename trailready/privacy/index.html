TrailReady Privacy Policy

TrailReady is a local-first outdoor trip planning and packing app operated by Korani Solutions AS. This policy explains how the current private beta accesses, uses, stores, and shares data.

Local-first beta Most app data is stored on the device, not in a Korani Solutions cloud account.
No ads or analytics SDK The current beta does not include advertising, analytics, crash-reporting, accounts, or cloud sync.

Who Operates The App

TrailReady is operated by Korani Solutions AS. For privacy, support, or data questions, contact Jonas.korani@gmail.com.

Data Stored On Device

TrailReady stores app content locally on the user's device. This can include trip plans, route details, gear inventory, packing selections, meal planning, notes, profile preferences, weather cache, and backup/import state.

Korani Solutions AS does not operate a user account system or cloud sync service for the current beta.

Location, Maps, Weather, Routes, And Fishing Data

When users choose features such as maps, route planning, place search, weather, fishing water, or fishing evidence, TrailReady may send the approximate place, coordinate, route, or query needed to external providers so those features can respond.

If current-location features are used, the app may ask for location permission through the operating system. Users can deny that permission, but location-based features may be limited.

Photos And Media

If a user adds photos to trips, gear, meals, notes, or profile areas, TrailReady asks for media library access through the operating system permission prompt. Selected images are used inside the app experience and may be included in user-created backups if the backup feature supports that content.

Backups And Sharing

TrailReady can create offline backup files or share links/codes. A person with that file, link, or code may be able to import the included trip, gear, packing, note, or route information. Only share backups with people you trust.

Analytics, Ads, Accounts, And Sale Of Data

The current beta build does not include an analytics SDK, crash-reporting SDK, advertising SDK, account system, or Korani Solutions cloud sync.

Korani Solutions AS does not sell TrailReady user data. Data may be shared only when needed to provide a feature the user requests, when the user chooses to share/export/backup data, or when required by law.

Data Retention And Deletion

Because the current beta is local-first, deleting the app or clearing app storage can remove local TrailReady data from the device. Export a backup before uninstalling if you want to keep your data.

Security

TrailReady is designed to keep beta data local where practical. Users are responsible for protecting their own device, backup files, and any links or codes they choose to share.

Policy Updates

This policy may be updated when TrailReady changes. The date at the top of this page shows the latest update.
